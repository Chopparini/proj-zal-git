O projekcie

Nasz projekt to aplikacja API stworzona z użyciem FastAPI. Umożliwia pobieranie kursów walut z oficjalnego API Narodowego Banku Polskiego (NBP).

Główne funkcje:

  • Główny Endpoint – dzisiejsze kursy USD i EUR
  • Integracja z zewnętrznym API (NBP)
  • Obsługa błędów i odpowiedzi JSON
